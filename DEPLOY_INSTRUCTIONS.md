## 🚨 Если что-то не работает:

1. **Ошибка сборки:** Проверь консоль для ошибок
   ```powershell
   npm run build 2>&1 | tail -100
   ```

2. **Компонент не видно:** Проверь импорт в page.tsx
   ```typescript
   import { DonateWidget } from '@/components/histo-helper/donate-widget'
   ```

3. **Стили не применяются:** Убедись что используются Tailwind классы
   
4. **Кнопки не работают:** Проверь `'use client'` в начале donate-widget.tsx
